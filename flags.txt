типы вывода на экран
+l - full info
+1 - одна колонка
+m - через запятую
+C - много колонок с сортировкой слева-направо сверху-вниз
++x - много колонок с сортировкой сверху-вниз слева-направо

тип вывода названия файла
B - octal type in any case: \012
b - C-escape code if can or octal: \n
++q - ????????
++v - печатает как есть
++w - печатает как есть (?)

дополнение к имени
+F - directory/ executable* link@ socet= whiteout% FIFO|
+p - directory/

тип времени
+c - последнее изменение статуса файла
++U - время создания файла
+u - время последнего доступа к файлу

скрытые файлы
+A - файлы с точкой в начале имени
+a - A + . ..

тип сортировки при выводе
+f - выводить как записаны на диск (нет сортировки)
+r - меняет направление выбранной сортировки на противоположное
+S - по размеру файлов
+t - сортировка по выбранному типу вывода времени

дополнительная инфа к l
+@ - дополнительные атрибуты ключей и размеров
+e - ACL связанный с файлом, если есть

дополнительная инфа или её игнор
+++g - убирает имя юзверя
+++h - меняет тип вывода размера файла с байтов на б, Кб, Мб, Гб, Тб
++i - i-node номер перед всей инфой о файле
+++n - вместо имени юзера ставит число
+++O - добавляет флаги файла всередине длинного вывода
o - убирает GroupID
++s - добавляет сверху "total:XXX" + перед инфой о файле выводит число 
+++T - меняет формат вывода даты-времени на более полный

extras
+G - цветной вывод имени файла
+R - рекурсивный вывод
d - папка == файл

+ - default
++ - 5(7) adds
+++ - 5 with l
++++ - more

===============================================================================

взаимодействие флагов
m - при комбинировании с -s не добавляет total
t - использует выбранный параметр времени для сортировки
@ и e - всегда используются с l, в комбинации всегда выводят аргументы @ а потом e
i и s - в комбинации всегда выводят аргументы i а потом s (total так и висит сверху)
i и s - влияют на вывод при C и x
f - по умолчанию врубает a и вырубает остальные сортировки (даже r)
n и g и o - по умолчанию врубают l
G - выключается если пишем в файл

СОРТИРОВКА:
t => S => f

===============================================================================

default flags by category

C - default to terminal
1 - default not to terminal

q - default to terminal
v - default not to terminal
w - default not to terminal (?)

c - default for time
